This guide walks through the basics of logging with Scribe, from your first log message to configuring categories and output.

## Your first log

After [installing Scribe](/installation), import it and log a message:

```swift
import Scribe

Log.info("App launched")
```

**Output:**
```text
2026-03-01 14:23:45.678+0000 ℹ️ [main.swift] App launched — main.swift:3
```

<Note>
  When you don't specify a category, Scribe uses the file name as the default category.
</Note>

## Define reusable categories

For production apps, define categories once and reuse them throughout your codebase:

```swift
import Scribe

extension LogCategory {
    static let app = LogCategory("App")
    static let auth = LogCategory("Auth")
    static let network = LogCategory("NetworkLayer")
    static let apiService = LogCategory("APIService")
}
```

Now use these categories in your logs:

```swift
Log.debug("Bootstrapping app", category: .app)
Log.info("User signed in", category: .auth)
Log.warn("Slow response", category: .network)
Log.error("Failed to decode payload", category: .apiService)
```

**Output:**
```text
2026-03-01 14:23:45.123+0000 🔍 [App] Bootstrapping app — AppDelegate.swift:42
2026-03-01 14:23:46.456+0000 ℹ️ [Auth] User signed in — AuthManager.swift:87
2026-03-01 14:23:47.789+0000 ⚠️ [NetworkLayer] Slow response — NetworkClient.swift:134
2026-03-01 14:23:48.012+0000 ❌ [APIService] Failed to decode payload — APIService.swift:56
```

## Configure minimum log level

Set a minimum log level to filter out less important messages:

```swift
// Only log messages at info level or higher
LogManager.shared.minimumLevel = .info

Log.debug("This won't appear", category: .app)
Log.info("This will appear", category: .app)
```

<Note>
  In production, set `minimumLevel` to `.info` or `.warning` to reduce noise and improve performance.
</Note>

## Filter by category

Restrict logging to specific categories during development:

```swift
let config = LogConfiguration(enabledCategories: [.network, .apiService])
LogManager.shared.configuration = config

Log.info("This won't appear", category: .app)
Log.info("This will appear", category: .network)
```

To allow all categories again:

```swift
let config = LogConfiguration(enabledCategories: nil)
LogManager.shared.configuration = config
```

## Use instance loggers with @Loggable

The `@Loggable` macro generates a `log` property automatically:

<CodeGroup>
```swift Basic usage
@Loggable
struct TokenManager {
    func refresh() {
        log.info("Refreshing token")
        // Category automatically uses type name: "TokenManager"
    }
}
```

```swift Custom category name
@Loggable("Auth")
struct TokenManager {
    func refresh() {
        log.info("Refreshing token")
        // Category is "Auth"
    }
}
```

```swift Existing category
extension LogCategory {
    static let auth = LogCategory("Auth")
}

@Loggable(category: .auth)
struct TokenManager {
    func refresh() {
        log.info("Refreshing token")
        // Uses .auth category
    }
}
```

```swift Static logger
@Loggable(style: .static)
struct APIClient {
    static func fetch() {
        log.info("Fetching data")
        // Static log property instead of instance
    }
}
```
</CodeGroup>

## Add a sink for testing

Sinks receive formatted log lines and can be used for test assertions or file output:

```swift
let sinkID = LogManager.shared.addSink { line in
    print("SINK:", line)
}

Log.info("Test message", category: .app)
// Logs to both os.log and your sink

// Remove the sink when done
LogManager.shared.removeSink(sinkID)
```

For async/await, use streams instead:

```swift
let stream = LogManager.shared.stream()

Task {
    for await line in stream {
        print("Log:", line)
    }
}
```

## Complete example

Here's a real-world example combining everything:

```swift
import Scribe

// 1. Define categories
extension LogCategory {
    static let app = LogCategory("App")
    static let network = LogCategory("Network")
}

// 2. Configure logging
func configureLogging() {
    #if DEBUG
    LogManager.shared.minimumLevel = .debug
    #else
    LogManager.shared.minimumLevel = .info
    #endif
    
    let config = LogConfiguration(
        enabledCategories: nil,  // Allow all categories
        includeTimestamp: true,
        includeEmoji: true
    )
    LogManager.shared.configuration = config
}

// 3. Use @Loggable for instance loggers
@Loggable
final class APIService {
    func fetchProfile() {
        log.api("GET /v1/profile")
        
        // Simulate API call
        DispatchQueue.main.asyncAfter(deadline: .now() + 0.5) {
            self.log.success("Profile fetched successfully")
        }
    }
}

// 4. Log in your app lifecycle
@main
struct MyApp: App {
    init() {
        configureLogging()
        Log.info("App launched", category: .app)
    }
    
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}
```
