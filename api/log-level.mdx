## Overview

The `LogLevel` enum represents the severity and domain of a log message. Log levels are organized into families for logical grouping and include emoji indicators, short codes, and OS log type mappings.

```swift
// Compare levels
if currentLevel >= .warning { ... }

// Filter by family
let networkLevels = LogLevel.levels(in: .networking)

// Parse from string
let level = LogLevel.parse("ERR") // Returns .error
```

## Helper methods

### levels(minimum:)

Returns all log levels at or above the specified minimum.

```swift
public static func levels(minimum: LogLevel) -> [LogLevel]
```

<ParamField path="minimum" type="LogLevel" required>
  The minimum level threshold
</ParamField>

<ResponseField name="return" type="[LogLevel]">
  Array of levels sorted by raw value
</ResponseField>

#### Example

```swift
let importantLevels = LogLevel.levels(minimum: .warning)
// Returns: [.warning, .error, .fatal, .success, .done, ...]
```

### parse(_:)

Parses a string into a `LogLevel`.

Accepts level names (e.g., "error"), short codes (e.g., "ERR"), or emojis (e.g., "❌").

```swift
public static func parse(_ string: String) -> LogLevel?
```

<ParamField path="string" type="String" required>
  The string to parse
</ParamField>

<ResponseField name="return" type="LogLevel?">
  The matching `LogLevel`, or `nil` if no match found
</ResponseField>

#### Example

```swift
if let level = LogLevel.parse("ERR") {
    print("Parsed level: \(level.name)") // "error"
}

if let level = LogLevel.parse("info") {
    print("Parsed level: \(level.emoji)") // "ℹ️"
}

if let level = LogLevel.parse("❌") {
    print("Parsed level: \(level.name)") // "error"
}
```

## OS integration

### osLogType

Maps this log level to the appropriate `OSLogType`.

```swift
public var osLogType: OSLogType { get }
```

<ResponseField name="osLogType" type="OSLogType">
  The corresponding OSLogType for use with Apple's unified logging system
</ResponseField>

#### Mappings

- `.trace`, `.debug`, `.print` → `OSLogType.debug`
- `.info`, `.notice`, `.success`, `.done` → `OSLogType.info`
- `.error` → `OSLogType.error`
- `.fatal` → `OSLogType.fault`
- All others → `OSLogType.default`

#### Example

```swift
let osType = LogLevel.error.osLogType
// Returns: OSLogType.error
```

## Comparison

Log levels are comparable based on their raw values:

```swift
if LogLevel.error > LogLevel.warning {
    print("Error is more severe than warning")
}

if currentLevel >= .info {
    print("Level is info or above")
}
```
